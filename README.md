# Chojema HTML Buddy

A tiny, dependency-free HTML WYSIWYG editor that runs entirely in your browser. Open or drag-and-drop an HTML file, edit it visually, and save it back out.

## Features

- **Open or drag-and-drop** any `.html` file — no upload, no server, everything stays local in your browser.
- **WYSIWYG editing** of the loaded document in place (layout, styles, and structure are preserved).
- **Live source panel** on the right, editable and kept in sync with the visual editor in both directions in real time. Drag the divider to resize it, or hide it to gain full width.
- **Formatting toolbar**: undo/redo, bold/italic/underline/strikethrough, keyboard-key (`<kbd>`) style, clear formatting, paragraph/heading/quote/code block styles, bulleted and numbered lists (with a choice of bullet/numbering styles), text alignment, text/background color.
- **Korean word-wrap protection** — a toggle that keeps Korean words from breaking mid-word at line wraps.
- **Table editing** — insert a table, insert/delete rows and columns, merge and split cells, and control border width and cell spacing.
- **Links** with an option to open in a new tab, and **images** from a local file or a URL.
- **Symbol menu** for quickly inserting common typographic symbols and arrows.
- **Save** the edited document back out as an HTML file.
- **Korean / English interface** — switch languages with the KO / EN buttons in the status bar; your choice is remembered.
- **No build step, no dependencies** — a single self-contained HTML file that works offline.

## Usage

1. Open `index.html` in any modern browser.
2. Drag an HTML file onto the window, or click "Choose File" / "Open File" to pick one.
3. Edit the document visually, using the toolbar or the live source panel — either side stays in sync with the other.
4. Click "Save" to download the edited HTML file.

This tool is best suited to touching up an **existing** HTML file rather than authoring a complex new document from scratch.

# Chojema HTML Buddy

브라우저에서 바로 동작하는, 의존성 없는 초경량 HTML WYSIWYG 편집기입니다. HTML 파일을 열거나 드래그앤드롭하면 화면에서 바로 시각적으로 편집하고, 다시 파일로 저장할 수 있습니다.

## 주요 기능

- **파일 열기 또는 드래그앤드롭**으로 `.html` 파일을 불러옵니다 — 서버 업로드 없이 브라우저 안에서만 동작합니다.
- 불러온 문서를 그 자리에서 **WYSIWYG로 편집**(레이아웃·스타일·구조가 그대로 유지됩니다).
- 우측의 **실시간 소스 패널**이 편집 화면과 양방향으로 실시간 동기화됩니다. 경계선을 드래그해 폭을 조절하거나, 숨겨서 편집 화면을 넓게 쓸 수 있습니다.
- **서식 툴바**: 실행 취소/다시 실행, 굵게/기울임/밑줄/취소선, 키보드 키(`<kbd>`) 모양 서식, 서식 지우기, 본문/제목/인용구/코드 블록 문단 서식, 글머리 기호·번호 매기기 목록(스타일 선택 가능), 정렬, 글자색·배경색.
- **한국어 줄바꿈 보호** — 한국어 단어가 줄바꿈 시 중간에서 잘리지 않도록 하는 토글 기능.
- **표 편집** — 표 삽입, 행/열 삽입·삭제, 셀 병합·분할, 테두리 굵기와 셀 간격 조절.
- 새 탭 열기 옵션이 있는 **링크 삽입**, 로컬 파일 또는 URL을 통한 **이미지 삽입**.
- 자주 쓰는 문장부호·화살표 기호를 바로 삽입하는 **기호 메뉴**.
- 편집한 문서를 HTML 파일로 **저장**.
- **한국어/영어 인터페이스** — 상태 표시줄의 KO / EN 버튼으로 전환할 수 있으며, 선택한 언어는 기억됩니다.
- **빌드 과정에 의존성 없음** — 단일 HTML 파일로 오프라인에서도 동작합니다.

## 사용법

1. 아무 최신 브라우저에서 `index.html`을 엽니다.
2. HTML 파일을 창에 드래그하거나, "파일 선택" / "파일 열기" 버튼으로 파일을 선택합니다.
3. 툴바나 실시간 소스 패널을 이용해 문서를 시각적으로 편집합니다 — 양쪽 어디를 편집해도 서로 동기화됩니다.
4. "저장" 버튼을 눌러 편집한 HTML 파일을 다운로드합니다.

이 도구는 복잡한 새 문서를 처음부터 작성하기보다는, **이미 존재하는** HTML 파일을 다듬는 용도로 사용하는 것이 가장 적합합니다.
